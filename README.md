# Terapêutica Clínica - Módulo Saúde Mental

Material didático em psicofarmacologia para o ciclo clínico de Medicina (UFSJ): aprendizagem baseada em casos clínicos.

🔗 Publicado em <https://henriquealvarenga.com/terapeutica-clinica>
📦 Repositório: <https://github.com/henriquealvarenga/terapeutica-clinica>

## Estrutura

```
terapeutica-clinica/
├── _quarto.yml         # configuração do site, navbar, tema
├── theme-editorial.scss # tema editorial (paper + brand orange)
├── styles.css           # @font-face self-hosted + overrides finos
├── styles.scss          # tema antigo (mantido como referência)
├── fonts/               # Inter, Playfair Display, JetBrains Mono (.woff2)
├── index.qmd            # capa
├── aula1.qmd            # Benzodiazepínicos (situações agudas)
├── aula2.qmd            # ISRS (depressão, ansiedade, TOC)
├── aula3.qmd            # Transtorno bipolar
├── aula4.qmd            # Esquizofrenia (antipsicóticos, clozapina)
├── como-estudar.qmd     # guia de uso de IA e fontes
├── creditos.qmd         # autor, licença, como citar
├── gabarito.qmd         # gabarito comentado (não vai para o repo público)
└── .github/workflows/publish.yml  # deploy automático no GitHub Pages
```
